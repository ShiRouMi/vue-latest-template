# vue-latest-template
vue 项目模板，使用最新脚手架搭建，结合个人经验与他人经验，整理的项目模板

- 数据转换器，减少前端模板中防御性代码
- 表单组件
- 自动生成 dll
- 封装 axios,拦截请求与响应
- storage 封装
- cookie 封装
- loading 封装
- 自定义拖拽指令
